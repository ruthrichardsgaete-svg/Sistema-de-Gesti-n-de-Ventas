Proyecto: Sistema de Gestión de Ventas

CREACIÓN Y MANIPULACIÓN DE TABLAS

La empresa comercial “Comercio Ya” necesita desarrollar un sistema básico
de gestión de ventas para registrar clientes, productos y transacciones en una
base de datos relacional. Actualmente, la empresa lleva su información en
planillas que generan errores frecuentes y dificultan el análisis posterior. El equipo
de tecnología solicita el desarrollo de una solución simple en SQL que permita
crear, poblar y consultar datos en un entorno confiable.

Objetivo; Desarrollar un sistema de gestión de ventas utilizando una base de datos
relacional, permitiendo almacenar, consultar y manipular información de clientes,
productos y ventas.
El objetivo es que el sistema esté completamente funcional, con su estructura
diseñada y sus consultas básicas operativas, utilizando únicamente los
contenidos de los manuales estudiados en el módulo.
